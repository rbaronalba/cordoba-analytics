# Córdoba CF Analytics Dashboard

Data analytics project analysing Córdoba CF performance in LaLiga Hypermotion.

## Features

- Advanced metrics analysis (xG, xGA)
- Tactical identity visualization
- League comparison
- Monte Carlo simulation for season projection
- Interactive dashboard

## Tech Stack

Python  
Pandas  
JavaScript  
Monte Carlo simulation  
Poisson goal model  

## Project Structure

dashboard → frontend visualization  
src → data pipeline & scrapers  
data → raw and processed datasets
